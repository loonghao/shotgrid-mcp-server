# 查询与可视化

查询任务安排并可视化工作量数据。

## 演示

![查询任务安排与工作量可视化](/01-query-projects-visualize-tasks.gif)

## 提示词

```
查询近一周的组员任务安排，工作量的负载率每天按照工时8小时来算，用 web 方式可视化显示
```

## 功能说明

1. 查询过去一周分配给团队成员的所有任务
2. 基于每天 8 小时工作标准计算工作量
3. 生成可视化 Web 报告，显示：
   - 每个团队成员的任务分布
   - 工作量百分比
   - 超负荷/低负荷指示器

## 使用场景

- 迭代规划和资源分配
- 团队产能分析
- 识别生产瓶颈
- 周报状态汇报
