# 企业微信集成

计算部门效率并发送报告到企业微信。

## 演示

![部门效率统计并发送企业微信](/04-department-efficiency-wecom.gif)

## 提示词

```
整理出部门的效率，将数据发送到企业微信，部门效率计算公式如下：效率 = 任务 bid / timelog 工时
```

## 功能说明

1. 从 ShotGrid 检索任务预估工时（bid）
2. 检索实际工时日志
3. 计算每个部门的效率比率
4. 格式化报告
5. 发送报告到企业微信

## 效率公式

```
效率 = 任务预估工时 / 实际工时日志
```

- **> 100%**：低于预算（高效）
- **= 100%**：符合预算
- **< 100%**：超出预算（需要关注）

## 使用场景

- 每日/每周效率报告
- 部门绩效跟踪
- 自动通知管理层
- KPI 监控和预警
