# TimeLog 统计

从 ShotGrid 查询并可视化工时日志数据。

## 演示

![TimeLog 统计与可视化](/03-timelog-statistics-visualize.gif)

## 提示词

```
查询 shotgrid 上的 timelog 数据并以 web 的方式可视化显示出来
```

## 功能说明

1. 从 ShotGrid 检索所有工时日志条目
2. 按用户、项目或任务汇总数据
3. 生成交互式 Web 图表，显示：
   - 每个用户的记录工时
   - 跨项目的时间分布
   - 每日/每周趋势

## 使用场景

- 时间跟踪分析
- 项目预算监控
- 资源利用率报告
- 账单和发票数据
