# Installation

ShotGrid MCP Server can be installed using pip, the Python package manager.

## Requirements

- Python 3.8 or higher
- ShotGrid API access
- ShotGrid Python API (shotgun-api3)

## Installation Steps

### 1. Install from PyPI

```bash
pip install shotgrid-mcp-server
```

### 2. Install from Source

For the latest development version, you can install directly from GitHub:

```bash
pip install git+https://github.com/loonghao/shotgrid-mcp-server.git
```

### 3. Development Installation

If you plan to contribute to the project, clone the repository and install in development mode:

```bash
git clone https://github.com/loonghao/shotgrid-mcp-server.git
cd shotgrid-mcp-server
pip install -e ".[dev]"
```

## ShotGrid API Installation

ShotGrid MCP Server requires the ShotGrid Python API. If you encounter issues with the automatic installation, you can install it manually:

```bash
pip install git+https://github.com/shotgunsoftware/python-api.git@v3.3.3
```

## Verifying Installation

To verify that ShotGrid MCP Server is installed correctly, run:

```bash
python -c "import shotgrid_mcp_server; print(shotgrid_mcp_server.__version__)"
```

This should print the version number of the installed package.

## Next Steps

Once you have installed ShotGrid MCP Server, proceed to the [Configuration](/configuration) section to set up your server.
