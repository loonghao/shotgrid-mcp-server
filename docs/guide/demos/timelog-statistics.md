# TimeLog Statistics

Query and visualize timelog data from ShotGrid.

## Demo

![TimeLog Statistics & Visualization](/03-timelog-statistics-visualize.gif)

## Prompt

```
Query timelog data from shotgrid and visualize it in web format
```

## What This Does

1. Retrieves all timelog entries from ShotGrid
2. Aggregates data by user, project, or task
3. Generates interactive web-based charts showing:
   - Hours logged per user
   - Time distribution across projects
   - Daily/weekly trends

## Use Cases

- Time tracking analysis
- Project budget monitoring
- Resource utilization reports
- Billing and invoicing data
