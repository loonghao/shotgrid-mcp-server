# Batch Operations

Batch create assets and assign tasks efficiently.

## Demo

![Batch Create Assets & Assign Tasks](/02-batch-create-assets-tasks-assign.gif)

## Prompt

```
Batch create the recommended hero characters in the shotgrid Demo:Animation 
project, categorize them as characters, use the FilmVFX-CharacterAsset task 
template, assign tasks to Yang Zhuo, with start and end dates set to next week
```

## What This Does

1. Creates multiple assets in batch
2. Categorizes them under the specified asset type
3. Applies task template to each asset
4. Assigns tasks to specified team member
5. Sets task schedule automatically

## Use Cases

- Project kickoff with multiple assets
- Bulk asset creation from concept art list
- Automated task assignment workflows
- Template-based task generation
